Variables

When using Make you'll make heavy use of variables and it's important to know when variable resolution (replacing the variable with its value) takes place.

A Make variable is created by an assignment statement, some examples: 

# immediate assignment:
hlq    := USER1

# delayed assignment
jclds   = $(hlq).JCL

# immediate assignment, referring to a variable with delayed assignment
someds := $(jclds)

hlq     = USER2

someds := $(jclds)
Note the assignment operator := which means it's an immediate assignment, or in other words, following this statement the variable myvar will contain the text 'some value'.
