Provided REXX execs

Along with the zmake source & binary there are a number of provided REXX execs to help you get started. You can find them here: https://github.com/btjassenskeiser/LWZMAKE/tree/master/SOURCE/EXEC.
They contain both REXX execs to be used in recipe CALL statements, and REXX execs to be used as functions.

REXX execs for recipe CALL statements

The following REXX execs can be used in recipe CALL statements:

ASMA Assemble a source by invoking ASMA90.
BIND Execute a DSN SYSTEM command followed by a DB2 BIND statement.
COBOL5 Compile a COBOL source by invoking IGYCRCTL (version 5 or up).
DFHEAP Translate CICS commands into regular assembler code by invoking DFHEAP1$
DSNHPC Pre-compile DB2 statements in a source by invoking DSNHPC
EQALANGX Create an EQALANGX file from an ADATA source by invoking EQALANGX
IEBCOPY Copy one or more members from one PDS(E) to another by invoking IEBCOPY
JUSTECHO Issue a REXX SAY with whatever parameter was passed JUSTECHO.
LKED Link-edit one or more objects into a load module by invoking IEWBLINK
SAFEDEL Safely delete one or more data sets and/or PDS(E) members
SUBMIT Submit a single JCL contained in the provided data set name or PDS(E) member
TOUCHMEM Set the last modified date+time to the current system date+time for one or more PDS(E) members
TSOCMD Execute a single TSO command
UNIXCMD Execute a single Unix System Services (USS) command
UPDCOPY Copies an input data set to an output data set and performs one or more search and replace actions during copying

REXX execs for ${function} statements

The following REXX execs can be used in ${function} statements:

APNDALL Append the first argument to each of the following arguments
APPEND Append the second argument to the first argument
LOWER Convert any uppercase characters in the argument(s) to lowercase
NEWDUMMY Create a new PDS(E) member with new ISPF stats to serve as a dummy member
PREPALL Prepend the first argument to each of the following arguments
PREPEND Prepend the second argument to the first argument
REPLACE Replace in the first argument any occurrence of the second argument with the third argument
SUBSTR Get a substring from the first argument, starting at the second argument and optionally limited to the length provided in a third argument
TEMPPDS Allocate a temporary PDS and return its data set name
UPPER Convert any lowercase characters in the argument(s) to uppercase
